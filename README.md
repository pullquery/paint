# Colors
CLI Color Painter with ANSI Escape Code

## Keys
|  KEY | COLOR   |
|-----:|:--------|
 | WASD | ↑ ← ↓ → |
|    ` | WHITE   |
|    1 | RED     |
|    2 | GREEN   |
|    3 | BLUE    |
|    4 | YELLOW  |
|    5 | MAGENTA |
|    6 | CYAN    |

## Scripts
```shell
go run colors # Run

go build colors # Build to binary
```
