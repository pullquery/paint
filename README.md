# Colors
CLI Color Painter with [ANSI Escape Code](https://en.wikipedia.org/wiki/ANSI_escape_code)

## Keys
|  KEY | ACTION  |
|-----:|:--------|
| WASD | ↑ ← ↓ → |
|    R | CLEAR   |
|    Q | EXIT    |
|    ` | WHITE   |
|    1 | RED     |
|    2 | GREEN   |
|    3 | BLUE    |
|    4 | YELLOW  |
|    5 | MAGENTA |
|    6 | CYAN    |


## Scripts
```shell
go run colors # Run

go build colors # Build to binary
```
